How to use icons in your slides:

Place the icon before your heading text using standard markdown image syntax:

# ![](/icons/rocket.svg) My Slide Title
## ![](/icons/chart-bar.svg) Column Header
### ![](/icons/lightbulb.svg) Section Title

Icons are automatically sized to match the heading text. Use black & transparent SVGs only for a clean look. Click any icon below to copy its markdown reference.
